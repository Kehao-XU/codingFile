# Markdown 文字格式调整指南

Markdown 是一种轻量级标记语言，使用简单的语法就能实现丰富的文字格式。以下是常用的文字格式调整方式：

## 一、标题

使用 `#` 符号可以创建不同级别的标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

显示效果：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 二、文本样式

### 1. 加粗
使用 `**` 或 `__` 包围文本：

```markdown
**这是粗体文本**
__这也是粗体文本__
```

显示效果：**这是粗体文本**

### 2. 斜体
使用 `*` 或 `_` 包围文本：

```markdown
*这是斜体文本*
_这也是斜体文本_
```

显示效果：*这是斜体文本*

### 3. 粗斜体
结合使用 `***` 或 `___` 包围文本：

```markdown
***这是粗斜体文本***
___这也是粗斜体文本___
```

显示效果：***这是粗斜体文本***

### 4. 删除线
使用 `~~` 包围文本：

```markdown
~~这是被删除的文本~~
```

显示效果：~~这是被删除的文本~~

### 5. 下划线
Markdown 本身不直接支持下划线，需要使用 HTML 标签：

```markdown
<u>这是带下划线的文本</u>
```

显示效果：<u>这是带下划线的文本</u>

## 三、列表

### 1. 无序列表
使用 `-`、`*` 或 `+` 作为列表标记：

```markdown
- 第一项
- 第二项
  - 子项1
  - 子项2
* 第三项
+ 第四项
```

显示效果：

- 第一项
- 第二项
  - 子项1
  - 子项2
* 第三项
+ 第四项

### 2. 有序列表
使用数字加 `.` 作为列表标记：

```markdown
1. 第一项
2. 第二项
   1. 子项1
   2. 子项2
3. 第三项
```

显示效果：

1. 第一项
2. 第二项
   1. 子项1
   2. 子项2
3. 第三项

## 四、引用

使用 `>` 符号创建引用：

```markdown
> 这是一段引用文本
> 
> 引用可以包含多行
>> 这是嵌套引用
```

显示效果：

> 这是一段引用文本
> 
> 引用可以包含多行
>> 这是嵌套引用

## 五、代码

### 1. 行内代码
使用 `` ` `` 包围代码片段：

```markdown
我使用 `print("Hello World")` 打印了问候语
```

显示效果：我使用 `print("Hello World")` 打印了问候语

### 2. 代码块
使用三个 `` ` `` 包围代码块，并可指定语言：

```markdown
```python
for i in range(5):
    print(f"第 {i+1} 次循环")
```
```

显示效果：

```python
for i in range(5):
    print(f"第 {i+1} 次循环")
```

## 六、链接

### 1. 行内链接
```markdown
[百度](https://www.baidu.com)
[GitHub](https://github.com "GitHub官网")  # 带标题
```

显示效果：[百度](https://www.baidu.com) 或 [GitHub](https://github.com "GitHub官网")

### 2. 引用链接
```markdown
[百度][1]
[GitHub][2]

[1]: https://www.baidu.com
[2]: https://github.com "GitHub官网"
```

显示效果：[百度][1] 或 [GitHub][2]

[1]: https://www.baidu.com
[2]: https://github.com "GitHub官网"

## 七、图片

### 1. 基本图片
```markdown
![图片描述](https://example.com/image.jpg)
```

### 2. 带标题的图片
```markdown
![图片描述](https://example.com/image.jpg "图片标题")
```

### 3. 引用式图片
```markdown
![图片描述][3]

[3]: https://example.com/image.jpg "图片标题"
```

## 八、分隔线

使用三个或更多的 `-`、`*` 或 `_` 可以创建分隔线：

```markdown
---
***
___
```

显示效果：

---

## 九、表格

使用 `|` 和 `-` 创建表格：

```markdown
| 姓名 | 年龄 | 性别 |
|------|------|------|
| 张三 | 25   | 男   |
| 李四 | 30   | 女   |
```

显示效果：

| 姓名 | 年龄 | 性别 |
|------|------|------|
| 张三 | 25   | 男   |
| 李四 | 30   | 女   |

### 表格对齐

使用 `:` 可以设置表格对齐方式：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:------|:--------:|-------:|
| 内容1 | 内容2    | 内容3  |
```

显示效果：

| 左对齐 | 居中对齐 | 右对齐 |
|:------|:--------:|-------:|
| 内容1 | 内容2    | 内容3  |

## 十、任务列表

使用 `- [ ]` 创建待办事项，使用 `- [x]` 创建已完成事项：

```markdown
- [x] 学习 Markdown 基本语法
- [ ] 练习创建表格
- [ ] 学习使用代码块
```

显示效果：

- [x] 学习 Markdown 基本语法
- [ ] 练习创建表格
- [ ] 学习使用代码块

## 十一、转义字符

使用 `\` 可以转义特殊字符：

```markdown
\*这不是斜体\*
\# 这不是标题
\[这不是链接\]
```

显示效果：

\*这不是斜体\*
\# 这不是标题
\[这不是链接\]

## 总结

以上是 Markdown 中常用的文字格式调整方式。Markdown 语法简单易学，掌握这些基本格式后，你就能创建美观易读的文档了。建议你多练习，很快就能熟练掌握！
